SAP CAPM Demo Application

This repository contains a demo application built with SAP CAP.

The application allows users to upload or select a text file. After the file is processed, the application parses the text, identifies individual words, and calculates:
	•	The total number of words
	•	The number of occurrences for each word
	•	The number of letters in each word

The processed results are then displayed in a clear and user-friendly format within the application interface.

This project is intended to demonstrate how text parsing and word analysis can be implemented using SAP CAPM technologies in a simple and practical scenario.
